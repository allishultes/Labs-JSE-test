## Installation
To get the code ready to run:
```
pip install -r requirements.txt
```
## Run the script
To run the script:
```
python main.py
```
This runs the `ques_B.py` and `ques_E.py` scripts.

The text files are saved to `/output`, and are included in the repository.

## Testing
To run tests:
```
python test_B.py
python test_E.py
```
